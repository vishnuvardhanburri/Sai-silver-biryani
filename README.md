# Sai Silver Dum Biryani

Static restaurant landing page for Sai Silver Dum Biryani.

## Run locally

```bash
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/`.

## Deploy to GitHub Pages

1. Push this repository to the `main` branch.
2. In GitHub, open `Settings -> Pages`.
3. Set `Source` to `GitHub Actions`.
4. The included workflow at `.github/workflows/deploy.yml` will publish the site automatically on every push to `main`.
